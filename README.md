# TepuyLog-Deductive-Rescue
Desarrollo de un motor de inferencia utilizando PyDatalog para resolver la "Operación Tepuy-Log". Ante el aislamiento de una expedición científica en Paraitepuy por crecidas de ríos, el sistema digitaliza la topografía del terreno y aplica reglas recursivas para determinar rutas terrestres seguras y transitables desde el Campamento Base.
